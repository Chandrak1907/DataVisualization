Chandra's Data Visualization Project

Techniques

Dataset consists of complaints logged on Consumer Financial Protection Bureau (CFPB.gov). Each comaplaint consits of information regarding complaint received regarding a financial product: Product, Sub Product, Issue, Subissue, State, Zipcode, Submitted via, Date Received, Date sent to company, Company, Company response, Timely response, Consumer Disputed. Objective of visualisation of this dataset is to answer below questions:
  • In which US state, there are more complaints?
  • Which is the most frequently complained service about?
  • Is there any correlation between type of complaints?
  • What is the trend of complaints so far, after establishing CFPB?
For this purpose, from original dataset different four different sets are derived and below techniques were implemented on CFPB data set:
  • Choropleth Plot:
    • From original dataset, state wise number of complaints for each financial product or service was derived. Rows without state name are ignored. As, some of the US states are having less population compared to the other states, this dataset is normalized by deriving number of complaints per one million people.In choropleth plot, total number of comaplints per one million people are shown. And also an histogram of distribution of total number of complaints per one million people is shown at the top of the US map.
    • Lie factor - After normalizing the dataset, this plot has less lie factor.
    • Data-ink ratio - All the ink was used to signify the distribution of data or color each state on map of the US.
    • Data-density - Data density of this map is normal. Instead of showing information for all 50 states of US map in a barchart, I have shown the information on map of the US.
  • Bubble Plot:
    • I had a hunch that as there is a correlation between mortgage complaints, credit card and debt collection related complaints. So, I used bubble plot to verify this and identify if any other correlation exists.
    • Lie-factor - This plot uses normalized dataset. Therefore, this plot has no lie factor.
    • Data-ink ratio - is high. Ink is used plot necessary information and legend only.
    • Data-density - Data density is normal.
  • Time Series plot:
    • Time series plot is drawn to identify the trends in the dataset. Small multiple dataset is drawn at the top. On clicking any of the small multiple plots, area plot corresponding to that small multiple is drawn below.
    • Lie-factor - This plot is a time series plot. There is no lie factor in this plot
    • Data-ink ratio - is a bit low. I have drawn area plots for aesthetic purpose. Therefore, data-ink ratio is less than that of a corresponding line plot.
    • Data-density - Data density is normal.
  • Parallel plot:
    • As this is an exploratory exerice to identfiy patterns in given dataset, parallel plot is drawn. Population statistics used for normalizing the original dataset is included in this plot.
    • Lie-factor - There is no lie factor in parallel plot. It represents original data.
    • Data-ink ratio - is high. Ink is used plot lines connecting related data points.
    • Data-density - Data density is normal.

Interactivity

  • Choropleth Plot - In choropleth plot below interactive features are implemented.
    • Brushing feature on histogram - If a bar with a particular value is selected, then corresponding area on US map will be highlighted.
    • Further ther is an option to drag the selection in brushing feature. Two rectancle likes will appear as soon as a bar is selected on histogram. This selection can be changed by dragging rectanles on either side.
    • Reset button is provided to clear the selection on histogram.
    • Color legend to indicate the numeric behind each color in the map is provided.
    • Tool tip on the US map - Tool tip functionality on map of the US provides state name and total number of complaints per one million people in that state.
    • When a state is clicked, number of comaplaints per one million people for each complaint is shown in a horizontal bar at the bottom of chart.
  • Bubble Plot -
    • Select boxes - In bubble plot, I provided select boxes to select variables for X-axis, Y-axis and Radius variable. All these variables should be different to generate plot.
    • Tool tip - Tool tip is shown to provide information regarding each bubble in plot.
    • Legend - Legend in the plot shows color code for each region. Each bubble is color coded with region.
  • Time series Plot -
    • Small multiple plots are drawn at the top of chart to give an overall view of trend of each complaint type over a period of time.
    • Selecting any of the multiple plot will show an area plot corresponding to that small multiple.
    • On area plot, zoom and brush functionalities are implemented.
    • Also, tool tip in area plot shows number of complaints and date.
  • Parallel plot -
    • Brushing feature - Brushing feature is implemented in this plot to highlight a range and to highlight corresponding data points.
    • A feature to move parallel plot axis is also implemented. This will be useful for analyzing data with respect to a feature.
    • Legend - Legend in the plot shows color code for each region. Each line in this plot is color coded with a color corresponding to a region.

Conclusion (per visualization -or- overall)

  • Choropleth plot:
    • Total number of complaints per one million people is more in Delware, Maryland, NewHampshire, New Jersey, Georgia and Florida. This information will be useful to focus efforts to improve serives.
  • Bubble plot:
    • My hunch regarding correlation between number of complaints regarding mortage, credit card and debt collection seems to be right.
    • States in North Central region seem to have less number of mortgage and credit card complaints compared to other states in the US.
  • Time Series plot:
    • From time series plot, it can be concluded that number of complaints against bank and its related services are constant across the time.
    • There is an increase in Consumer product, Credit reporting and prepaid and student related complaints
  • Parallel plot:
    • All high populace states are having high number of complaints.
    • Number of complaints registed in states in Western region are less compared to other states. One of the reasons could be the service offered by financial institutions are better in this region.
  • Overall Conclusions & Scope for further work:

There are complaints against 2801 financial institutions in this dataset. Therefore, it will not be possible to ascertain the performance of a financial institution in a single visualization across all geographies. This is an attempt to look at the overall nature of complaints in the US after instituting CFPB
