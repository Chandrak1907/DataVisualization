Data Cleaning

  • Movies dataset in ggplot2 had 58,788 rows. This dataset contains movie information and user ratings from IMDB.com. This dataset contains below fields:
    • title. Title of the movie.
    • year. Year of release.
    • budget. Total budget (if known) in US dollars
    • length. Length in minutes.
    • rating. Average IMDB user rating.
    • votes. Number of IMDB users who rated this movie.
    • r1-10. Multiplying by ten gives percentile (to nearest 10%) of users who rated this movie a 1.
    • mpaa. MPAA rating.
    • action, animation, comedy, drama, documentary, romance, short. Binary variables representing if movie was classified as belonging to that genre
    This dataset has two problems.
    • There are many movies with many Genre values or no Genre values.
    • Movie budget was not available for many movies.
  • During pre-processing, movies with only Genre were considered. And, movies for which no budget column was specificed are removed. After applying these two rules, dataset now has 2543 movies.
  • Genre column was generated Tableau using calculated field feature.
  • Budget is a numerical value in original dataset. This is converetd to categorical variable with 6 values. i.e."Low to Quantile1", "Quantile1 to Median", "Median to Mean", "Mean to Quantile 3", "Quantile 3 to Highest value, using calcualted feature function in Tableau.

Approach

  • I chose to plot below three charts from cleaned data set:
    • Bar Chart - Genre wise movie number of movies in cleaned dataset. This chart gives information regarding which Genre movies were prodiced most so far.
    • Bar Chart - Average Movie Rating vs Budget Category wise I plotted this chart to see whether any correaltion exists between movie budget and average viewer rating.
    • Line plot - Average length of movies per year I wanted to check, whether movie lengths are increasing or decreasing for each genre.

Dataset preparation

  • I mainly used R for data set preperation for each of the charts. R code is available on Github in file "movies.R"
  • I used Python to generate JSON files required for d3.
