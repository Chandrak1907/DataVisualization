Type of data in the dataset

Given dataset contains 3.7 M rows of all complaints received about financial products and services. From this dataset, numerical data can be generated

Reason for chosing this dataset

This dataset contains below information regarding complaint received regarding a financial product: Product, Sub Product, Issue, Subissue, State, Zipcode, Submitted via, Date Received, Date sent to company, Company, Company response, Timely response, Consumer Disputed

Main reason for chosing this dataset is presence of State code and zip code. I derived below statistics from this data set:
  • Statewise number of complaints
  • Zipcode wise number of complaintsProject
  • Time Series Data of complaints - Financial Product wise
I am planning to plot below as part of final deliverables:
  • Heat map of count of financial product wise compltains Statewise
  • Heat map of count of financial product wise compltains Statewise on US map
  • Time Series plot of financial product wise complaints
  • I will aim to do zip code wise plot of count of financial product wise compltains
